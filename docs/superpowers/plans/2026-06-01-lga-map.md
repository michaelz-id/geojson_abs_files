# LGA Boundaries Interactive Map — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a single `index.html` file that renders Australian LGA boundaries from GeoJSON using Leaflet, coloured by state, with hover popups and a working search bar.

**Architecture:** Single static HTML file; all logic in an inline `<script>` block; GeoJSON fetched at runtime via `fetch()`; served by a local HTTP server (no `file://` — browsers block cross-origin fetch on that protocol).

**Tech Stack:** HTML5, CSS3, Leaflet 1.9.4 (CDN), vanilla JS (ES2020), Python `http.server` for local dev

```javascript
const DEFAULT_COLOUR = { fill: '#95a5a6', border: '#bdc3c7' };
const HIGHLIGHT_FILL = '#f39c12';

const STATE_LABELS = {
  '1': 'NSW', '2': 'VIC', '3': 'QLD', '4': 'SA',
  '5': 'WA',  '6': 'TAS', '7': 'NT',  '8': 'ACT'
};

function styleFeature(feature) {
  const code = String(feature.properties.state_code);
  const colour = STATE_COLOURS[code] || DEFAULT_COLOUR;
  return {
    fillColor: colour.fill,
    color: colour.border,
    weight: 1.2,
    fillOpacity: 0.85,
    opacity: 1
  };
}

let geoJsonLayer;

fetch('lga_boundaries.geojson')
  .then(r => r.json())
  .then(data => {
    document.getElementById('loading').remove();
    document.getElementById('lga-count').textContent =
      `${data.features.length.toLocaleString()} LGAs · Australia`;

    geoJsonLayer = L.geoJSON(data, {
      style: styleFeature
    }).addTo(map);
  })
  .catch(() => {
```

## Task 4: Hover popup

- [ ] **Step 1: Add onEachFeature function**

Insert this function directly before the `fetch(` line:

```javascript
function onEachFeature(feature, layer) {
  const { lga_name, lga_code, state_name, area_sqkm } = feature.properties;
  const area = Number(area_sqkm).toLocaleString('en-AU', { maximumFractionDigits: 0 });
  layer.bindPopup(
    `<strong>${lga_name}</strong><br/>LGA Code: ${lga_code}<br/>${state_name} · ${area} km²`,
    { closeButton: false, autoPan: false }
  );

  layer.on('mouseover', function () {
    this.setStyle({ fillColor: HIGHLIGHT_FILL, weight: 2 });
    this.openPopup();
  });
  layer.on('mouseout', function () {
    geoJsonLayer.resetStyle(this);
    this.closePopup();
  });
}
```

## Task 5: State legend

**Files:**
- Modify: `index.html` — add `buildLegend` function and call it after layer is added

- [ ] **Step 1: Add buildLegend function**

Insert this function directly before the `fetch(` line (after `onEachFeature`):

```javascript
function buildLegend() {
  const legend = L.control({ position: 'bottomright' });
  legend.onAdd = function () {
    const div = L.DomUtil.create('div', 'legend');
    const items = Object.entries(STATE_LABELS)
      .map(([code, label]) => {
        const fill = (STATE_COLOURS[code] || DEFAULT_COLOUR).fill;
        return `<div class="legend-item">
          <div class="legend-swatch" style="background:${fill}"></div>${label}
        </div>`;
      })
      .join('');
    div.innerHTML = `<h4>State / Territory</h4><div class="legend-grid">${items}</div>`;
    return div;
  };
  legend.addTo(map);
}
```

## Task 6: Search functionality

**Files:**
- Modify: `index.html` — add `initSearch` function and call it after layer is added

- [ ] **Step 1: Add initSearch function**

Insert this function directly before the `fetch(` line (after `buildLegend`):

```javascript
function initSearch(layers) {
  const input = document.getElementById('search');
  input.addEventListener('input', function () {
    const query = this.value.trim().toLowerCase();
    if (!query) {
      map.fitBounds(AUSTRALIA_BOUNDS);
      return;
    }
    const match = layers.find(
      l => l.feature.properties.lga_name.toLowerCase().includes(query)
    );
    if (match) {
      map.fitBounds(match.getBounds());
      match.openPopup();
    }
  });
}
```

- [ ] **Step 2: Call initSearch after buildLegend**

```javascript
geoJsonLayer = L.geoJSON(data, {

```
